Canon PIXMA E3370 Multi-function WiFi Color Inkjet Printer (Borderles Printing) (Black, Ink Cartridge)

Key features:

  • Output: Color
  • WiFi | USB
  • Print Speed Mono A4: 8 ipm | Print Speed Color A4: 4 ipm
  • Supported OS: Windows,macOS, Linux

Descrption:

Featuring an up to 3.81 cm LCD display, the Canon PIXMA E3370 Printer lets you effortlessly set the size and number of copies. Besides, with its easy-to-understand icons, this printer lets you keep an eye on its operational and wireless connection status. Compatible with Canon Print Inkjet/ SELPHY, and Easy-PhotoPrint Editor apps, this printer makes it hassle-free to wirelessly print using your smartphone, tablet, and laptop.
